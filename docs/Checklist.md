# Weekly Checklist (v0)

## Tuesday (Intake)
- [ ] Collect worker availability + preferences
- [ ] Update `data/templates/availability.csv`
- [ ] Confirm worker roster changes (new hires, quals)
- [ ] Update `data/templates/workers.csv` if needed
- [ ] Update OT totals if tracking (`data/templates/ot_totals.csv`)

## Wednesday (Planning)
- [ ] Confirm work orders, priorities, critical path
- [ ] Update `data/templates/work_orders.csv`
- [ ] Run OTRAK allocation (v0)
- [ ] Review draft plan + flagged exceptions
- [ ] Apply overrides with reasons
- [ ] Export final plan (CSV + web view)

## After
- [ ] Log changes/overrides for audit
- [ ] Archive weekly files
